Specs

The Specs section hosts URS, FS, and NFS documents following a GAMP-inspired structure, along with DS documents for each design chapter. All files are in Markdown for straightforward versioning and review.

An index file links to every specification. Each DS also has a matching theory page with an SVG diagram in the Theory section.
