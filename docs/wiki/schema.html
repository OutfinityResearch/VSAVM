Schema

A schema captures a recurring intent and provides a structured frame with typed slots. It is the bridge between surface language and executable programs.

Schemas emerge from repeated patterns and are refined through execution feedback. This allows new phrasing to map to stable program structures.

In VSAVM, schemas enable compact query compilation and reduce the search space for candidate programs.
