Minimum Description Length (MDL)

MDL is a compression principle that prefers models which explain data with minimal description length. It encourages structures that are both compact and predictive.

In VSAVM, MDL favors macro programs that repeatedly compress common patterns. This stabilizes the program library and reduces prediction surprise.

MDL also acts as a guardrail against uncontrolled complexity, preventing the system from growing brittle or overly specific rules.
