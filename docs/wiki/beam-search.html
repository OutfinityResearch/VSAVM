Beam search

Beam search maintains a limited set of candidate branches in a search process. It is a pragmatic compromise between exhaustive exploration and greedy choice.

In VSAVM, beam search appears in query compilation and in bounded closure to keep reasoning tractable while still exploring multiple interpretations.

The beam width becomes part of the budget contract, making the scope of exploration explicit and configurable.

External link: https://en.wikipedia.org/wiki/Beam_search.
