Fact store

The fact store is the canonical memory of assertions. Each assertion is mapped to a stable identifier, allowing conflicts to be detected across paraphrases.

Context metadata separates local theories and prevents unrelated sources from collapsing into a single inconsistent pool.

The fact store is central to bounded closure because it provides the substrate for both derivation and contradiction checks.
