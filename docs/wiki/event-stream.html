Event stream

The event stream is the canonical representation for all inputs. Each event carries a discrete payload and structural context, making segmentation and scope explicit.

This representation unifies text, audio, and visual inputs into a single processing pipeline. It also enables deterministic expansion from macro units to lexical tokens.

By emphasizing structure, the event stream supports consistency checking and localized reasoning inside the VM.
