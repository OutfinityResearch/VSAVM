Large Language Model (LLM)

A classical LLM optimizes next-token prediction. VSAVM preserves the LLM-like interface but conditions prediction on explicit VM state derived by execution.

This design keeps the familiar user experience while enabling consistency checks and traceable reasoning beneath the surface.

The difference is not in the interface, but in the internal contract that governs what the model is allowed to claim.

External link: https://en.wikipedia.org/wiki/Large_language_model.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
