Context and scope

Context defines where a statement applies and where it can be checked for contradiction. Scope separates local theories and prevents unrelated sources from colliding.

In VSAVM, scope is derived from structural separators and is carried through execution. This makes consistency checks sensitive to source boundaries.

The result is a system that can reason conditionally across sources without collapsing them into a single inconsistent base.
