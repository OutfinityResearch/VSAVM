Macro token

A macro token is a compressed unit derived from recurring phrase patterns. It reduces entropy at the phrase level while remaining reversible into elementary tokens.

This reversibility is critical because evaluation and decoding depend on deterministic expansion. Macro tokens improve efficiency without sacrificing traceability.

In VSAVM, macro tokens also anchor schema discovery and accelerate retrieval by forming stable discrete units.
