Operational correctness

Correctness is operational and budgeted. The system is allowed to state only what remains consistent under a bounded closure of rules and executions, which makes the reasoning effort explicit and measurable.

This requires a canonical fact language with explicit negation. By mapping statements to stable identifiers, the VM can detect contradictions even when surface wording differs.

The closure process is controlled by depth, step, and branching limits. These limits define how far the system can explore, and therefore how robust a conclusion can be claimed.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
