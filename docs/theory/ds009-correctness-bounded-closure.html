Operational correctness

Correctness is operational and budgeted. A statement is accepted only if bounded closure does not reveal contradictions within configured limits, making the effort explicit and measurable.

A canonical fact language is mandatory. By mapping surface statements to stable identifiers and explicit negation, the VM can detect contradictions even when wording differs.

Closure is controlled by depth, step, and branching limits. These limits define how far the system can explore and therefore how robust a conclusion can be claimed.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
