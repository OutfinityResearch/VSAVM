Learning to execute

Training is framed as a predictive task that nudges the system to discover executable patterns. The outer loop optimizes next-phrase prediction, while the inner loop searches for programs that compress repeated patterns into reusable procedures.

The NL to query compiler emerges from this pressure. When repeated questions appear, a compact program that explains them becomes cheaper under MDL, and thus more attractive than a purely surface-level continuation.

Consolidation is explicit. Programs that repeatedly win are elevated to macro instructions, creating a library of stable behaviors that the system can reuse instead of re-deriving each step.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
