Controlled generation

Generation is treated as a constrained decision process. Candidate phrases are proposed by a discrete language model and filtered through schema constraints and VM state to remain aligned with execution.

Before acceptance, each candidate is simulated. A local bounded closure is run to detect conflicts, turning correctness into a gating step rather than a post-hoc correction.

When the user asks for deeper analysis, the system increases search budget and explores more of the state space. If conflicts persist, responses become conditional or explicitly indeterminate.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
