Controlled generation

Generation is framed as a constrained decision. Candidate phrases are proposed by a discrete language model and then filtered through schema constraints and VM state. This keeps output aligned with the executable interpretation of the input.

Each candidate is simulated to test its impact on the current state. A local bounded closure is run to detect conflicts before the phrase is accepted, turning correctness into a gating mechanism rather than a post-hoc patch.

Deeper analysis is implemented by expanding the search budget. When the exploration limit increases, the system does more work instead of pretending to be certain, and it can explicitly mark results as conditional when necessary.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
