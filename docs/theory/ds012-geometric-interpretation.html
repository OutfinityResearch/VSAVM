State-space geometry

The conceptual geometry of VSAVM is defined by the VM state space. Each instruction is a transition, and reasoning is a path through this graph under explicit constraints.

VSA adds a secondary geometry of similarity that speeds up search but does not determine truth. Similar phrases map to nearby hypervectors, yet the VM decides whether a mapping is valid by executing it.

Larger budgets reveal deeper regions of the state graph. Thinking more therefore means exploring more transitions, not merely generating longer text.

VSAVM is an Axiologic Research experiment within the Achilles project. This static documentation is written in clear academic English for engineers.
