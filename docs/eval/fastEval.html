FastEval

Rapid VSAVM Evaluation Suite - Lightweight tests for core AI capabilities

üöÄ Quick Start

FastEval provides deterministic, lightweight tests that run in seconds to minutes, giving rapid feedback on VSAVM's core capabilities.

Installation & Setup

Get started with FastEval in under 2 minutes:

# Run all evaluations (from repo root) node evals/run.mjs # Run a specific category node evals/run.mjs --category reasoning # Run with verbose output node evals/run.mjs --verbose

Available Commands

  • node evals/run.mjs - Run complete evaluation suite
  • node evals/run.mjs --category rule-learning - Rule learning tests
  • node evals/run.mjs --category compression - Compression tests
  • node evals/run.mjs --category reasoning - Reasoning tests
  • node evals/run.mjs --category query-response - Query response tests
  • node evals/run.mjs --json - Emit JSON-only results

‚öôÔ∏è Configuration

FastEval is highly configurable to adapt to different testing scenarios and performance requirements:

{ "timeouts": { "rule_learning": 30000, "compression": 15000, "reasoning": 10000 }, "thresholds": { "rule_accuracy": 0.90, "compression_ratio": 0.50, "reasoning_consistency": 0.95 }, "data_sizes": { "small": 100, "medium": 1000, "large": 10000 } }

Threshold Justification

Rule Accuracy: 90%

Based on symbolic AI benchmarks where deterministic patterns should be learned with high accuracy on synthetic data.

Compression: 50%

Derived from information theory bounds on structured data with repetitive patterns.

Consistency: 95%

Logic programming systems achieve near-perfect consistency on well-formed knowledge bases.
